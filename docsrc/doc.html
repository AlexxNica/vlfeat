VLFeat documentation has three parts:

  • Matlab help. VLFeat Matlab commands embed documentation that can be viewed by Matlab help command (for instance, type help vl_sift to read the vl_sift command documentation). For convenience, the same documentation is also available in HTML format.

  • C API documentation. VLFeat source code embeds extensive documentation. This includes descriptions of all algorithms which can be useful even if you do not plan to use the C library directly.

  • Man pages. The command line utilities bundled with VLFeat are described by relative Unix man pages. These are found in the src/ subdirectory, or can be viewed in HTML format.

To get started, the tutorials can be useful.
